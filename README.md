
# OBRIO Test task


## Tech Stack

Next.js, Redux, SCSS, Typescript, Feature-Sliced Design methodology



## Run Locally

Clone the project

```bash
  git clone https://github.com/Remasterer/questionnaire
```

Go to the project directory

```bash
  cd questionnaire
```

Install dependencies

```bash
  npm install
```

Start the appp

```bash
  npm run dev
```


## Build and start

```bash
  npm run build && npm run start
```
